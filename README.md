# Analyzing Tool Cost

This project implements an end-to-end ELT data pipeline for analyzing tool costs in relation to production volume.

## Description
The goal of this project is to provide insights into how production impact tool costs and to identify the most cost-intensive tools.

The pipeline includes:

- Extracting data from multiple Excel files located in different directories
- Loading raw data into Power BI
- Transforming data using Power Query and DAX (calculated columns and measures)
- Building a relational data model
- Creating interactive dashboards

## Key Features
- Analysis of the relationship between tool cost and production volume
- Pareto analysis to determine which tools generate the majority of costs
- Time-based cost analysis

## Technologies Used
- Excel
- Power BI
- DAX (Data Analysis Expressions)
